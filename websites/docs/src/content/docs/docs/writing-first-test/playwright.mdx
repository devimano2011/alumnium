Let's see how we can write a simple test that would use [Playwright][3] to open DuckDuckGo, search for a Mercury element, ensure that search results are loaded, include the Wikipedia article about the element, and check that its symbol is presented correctly on the page.

## Install Dependencies

Make sure you have Alumnium installed in your project.

<LanguageContent lang="python">

```bash title="Installing dependencies"
$ pip install alumnium pytest-playwright
$ playwright install chromium
```

</LanguageContent>

<LanguageContent lang="typescript">

```bash title="Installing dependencies"
$ npm install alumnium
```

Now initialize Playwright in your project if you haven't done it yet. Leave all the options as default and remove example test after initialization.

```bash title="Initializing Playwright"
$ npm init playwright
$ rm tests/example.spec.ts
```

</LanguageContent>

## Setup Browser

Start by creating a test file for your tests and instantiating a browser instance:

<LanguageContent lang="python">

```python title="test_search.py"
from playwright.sync_api import Page


def test_search(page: Page):
    page.goto("https://duckduckgo.com")
```

Now run the test. You should see a Chrome browser window with DuckDuckGo opened.

```bash title="Running test"
$ pytest --quiet --headed
...
1 passed in 1.32s
```

</LanguageContent>

<LanguageContent lang="typescript">

```typescript title="tests/search.spec.ts"
import { test, expect } from "@playwright/test";

test("should search", async ({ page }) => {
  await page.goto("https://duckduckgo.com");
});
```

Now run the test. You should see a Chrome browser window with DuckDuckGo opened.

```bash title="Running test"
$ npx playwright test --project chromium --headed

Running 1 test using 1 worker
  1 passed (4.0s)
```

</LanguageContent>

## Setup Alumnium

Now let's add the code that would instantiate Alumnium using the browser. We can use a test fixture to ensure proper setup and teardown of the Alumnium instance:

<LanguageContent lang="python">

```python title="test_search.py" ins={1, 3, 6-10} ins="al: Alumni"
from alumnium import Alumni
from playwright.sync_api import Page
from pytest import fixture


@fixture
def al(page: Page):
    al = Alumni(page)
    yield al
    al.quit()


def test_search(al: Alumni, page: Page):
    page.goto("https://duckduckgo.com")
```

The test should still work fine, let's re-run it to make sure:

```bash title="Running test"
$ pytest --quiet --headed
...
1 passed in 7.81s
```

</LanguageContent>

<LanguageContent lang="typescript">

```typescript title="tests/index.ts"
import { test as base } from "@playwright/test";
import { Alumni } from "alumnium";

export const test = base.extend<{ al: Alumni }>({
  al: async ({ page }, use) => {
    const al = new Alumni(page);
    await use(al);
    await al.quit();
  },
});

export { expect } from "@playwright/test";
```

```typescript title="tests/search.spec.ts" del={1} ins={2} ins="al"
import { test, expect } from "@playwright/test";
import { test, expect } from ".";

test("should search", async ({ page, al }) => {
  await page.goto("https://duckduckgo.com");
});
```

The test should still work fine, let's re-run it to make sure:

```bash title="Running test"
$ npx playwright test --project chromium --headed

Running 1 test using 1 worker
  1 passed (8.2s)
```

</LanguageContent>

If you are getting errors about missing API keys, make sure you [configured][1] Alumnium and try again!

## Add Actions

Now let's add some actions that Alumnium should **do** on the page. Our test needs to search for a Mercury element, so let's use this exact command:

<LanguageContent lang="python">

```python title="test_search.py" ins={15}
from alumnium import Alumni
from playwright.sync_api import Page
from pytest import fixture


@fixture
def al(page: Page):
    al = Alumni(page)
    yield al
    al.quit()


def test_search(al: Alumni, page: Page):
    page.goto("https://duckduckgo.com")
    al.do("search for 'Mercury element' and press Enter")
```

Run the test, you should now see "Mercury element" typed in the search box and the page with results loaded.

```bash title="Running test"
$ pytest --quiet --headed
...
1 passed in 11.34s
```

</LanguageContent>

<LanguageContent lang="typescript">

```typescript title="tests/search.spec.ts" ins={5}
import { test, expect } from ".";

test("should search", async ({ page, al }) => {
  await page.goto("https://duckduckgo.com");
  await al.do("search for 'Mercury element' and press Enter");
});
```

Run the test, you should now see "Mercury element" typed in the search box and the page with results loaded.

```bash title="Running test"
$ npx playwright test --project chromium --headed
...
```

## Add Verifications

The next step is to add some verifications that Alumnium should **check** on the page. We are going to add two of them, one that checks that the page title contains the search keyword and one to see if the Wikipedia article is present in the results.

Common wisdom says to _never trust a test you haven't seen fail_. Let's add the first verification and see it fail!

<LanguageContent lang="python">

```python title="test_search.py" ins={16}
from alumnium import Alumni
from playwright.sync_api import Page
from pytest import fixture


@fixture
def al(page: Page):
    al = Alumni(page)
    yield al
    al.quit()


def test_search(al: Alumni, page: Page):
    page.goto("https://duckduckgo.com")
    al.do("search for 'Mercury element' and press Enter")
    al.check("page title contains Aluminium word")
```

Now let's run our test:

```bash title="Running test" wrap {4}
$ pytest --quiet --headed

...

...
1 failed in 25.21s
```

</LanguageContent>

Now let's run our test:

Our test failed as we expected and provided a meaningful explanation of what went wrong.

## Add Data Retrieval

Finally, let's verify some data that Alumnium can **get** from the page. We are going to ensure the Mercury element card shows its symbol correctly.

Let's add a failing verification first:

<LanguageContent lang="python">

```python title="test_search.py" ins={18,19}
from alumnium import Alumni
from playwright.sync_api import Page
from pytest import fixture


@fixture
def al(page: Page):
    al = Alumni(page)
    yield al
    al.quit()


def test_search(al: Alumni, page: Page):
    page.goto("https://duckduckgo.com")
    al.do("search for 'Mercury element' and press Enter")
    al.check("page title contains Mercury word")
    al.check("search results contain Wikipedia articles")
    symbol = al.get("chemical symbol")
    assert symbol == "Al"
```

Time to re-run the test:

```bash title="Running test" {3-6}
$ pytest --quiet --headed
...
E       AssertionError: assert 'Hg' == 'Al'
E
E         - Al
E         + Hg
...
1 failed in 49.14s
```

</LanguageContent>

<LanguageContent lang="typescript">

```typescript title="tests/search.spec.ts" ins={8-9}
import { test, expect } from ".";

test("should search", async ({ page, al }) => {
  await page.goto("https://duckduckgo.com");

  await al.check("page title contains Mercury word");

  const symbol = await al.get("chemical symbol");
  expect(symbol).toBe("Al");
});
```

Time to re-run the test:

The test failed as expected, let's fix it and re-run to make sure it's passing:
